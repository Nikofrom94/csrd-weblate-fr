Maîtrise des Machines (8 points d'Intellect) #Ability #SensesAbility #High #Core

Vous pouvez contrôler les fonctions d'une machine avec laquelle vous vous êtes lié en utilisant Liaison machine, intelligente ou autre. De plus, si vous utilisez une action pour vous concentrer sur une machine, vous êtes conscient de ce qui se passe autour d'elle (vous voyez et entendez comme si vous étiez à côté d'elle, quelle que soit la distance dont vous vous trouvez). Vous devez toucher la machine pour créer le lien, mais après, il n'y a aucune limitation de portée. Ce lien dure une semaine. Vous ne pouvez établir de liaison qu'avec une seule machine à la fois.
