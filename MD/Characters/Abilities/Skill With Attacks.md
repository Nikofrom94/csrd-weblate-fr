Compétence avec les attaques #Ability #Mid #Core

Choisissez un type d'attaque dans lequel vous n'êtes pas déjà entraîné : frappe légère, lame légère, distance légère, frappe moyenne, lame moyenne, portée moyenne, frappe lourde, lame lourde ou distance lourde. Vous êtes entraîné aux attaques utilisant ce type d'arme. Vous pouvez sélectionner cette capacité plusieurs fois.
