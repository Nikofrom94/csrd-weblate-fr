Télépathie machine (3 points d'intelligence) #Ability #InformationAbility #Mid #Core

Vous pouvez lire les pensées superficielles d'une machine à courte portée, même si la machine ne le souhaite pas. Vous devez pouvoir voir la machine. Une fois le contact établi, vous pouvez lire les pensées de la cible pendant une minute maximum. Si vous ou la cible vous déplacez hors de portée, la connexion est rompue. Si vous possédez la capacité de lecture mentale, lorsque vous appliquez l'effort à la télépathie machine, vous gagnez un niveau d'effort gratuit.
