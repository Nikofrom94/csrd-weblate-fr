Pied Léger (1+ points de Célérité) #Ability #Low #Core

Vous pouvez vous déplacer sur une courte distance dans le cadre d'une autre action. Vous pouvez vous déplacer sur une longue distance pendant toute votre action pendant un tour. Si vous appliquez un niveau d'Effort à cette capacité, vous pouvez vous déplacer sur une longue distance et effectuer une attaque pendant toute votre action pendant un tour, mais l'attaque est gênée. .
