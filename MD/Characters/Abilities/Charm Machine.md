Charmer une Machine (2 points d'Intellect) #Ability #ControlAbility #Low #Core

Vous convainquez une machine inivntelligente de vous « aimer ». Une machine qui vous aime a 50 % moins de chances de fonctionner si cette fonction risque de vous nuire. Ainsi, si un ennemi tente de faire exploser près de vous une bombe contrôlée par un détonateur qui vous aime, il y a 50 % de chances qu'elle n'explose pas.
