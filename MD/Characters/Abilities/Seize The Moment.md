Saisissez l'instant (4+ points de Célérité) #Ability #Mid #Core

Si vous réussissez un jet de défense de Célérité pour résister à une attaque, vous gagnez une action. Vous pouvez utiliser l'action immédiatement même si vous avez déjà joué un tour dans le tour. Vous n'effectuez aucune action lors du tour suivant, sauf si vous appliquez un niveau d'Effort lorsque vous utilisez Saisissez l'instant. .
