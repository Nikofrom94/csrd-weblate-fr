Explosion de froid (5+ points d'Intellect) #Ability #High #Core

Vous émettez une explosion de froid dans toutes les directions, jusqu'à courte portée. Tous ceux qui se trouvent dans la rafale (sauf vous) subissent 5 points de dégâts. Si vous appliquez Effort pour augmenter les dégâts plutôt que pour faciliter la tâche, vous infligez 2 points de dégâts supplémentaires par niveau d'Effort (au lieu de 3 points) ; les cibles dans la zone subissent 1 point de dégâts même si vous échouez au jet d'attaque.
