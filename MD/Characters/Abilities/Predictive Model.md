Modèle prédictif (2+ points d'Intellect) #Ability #InformationAbility #Low #Core

Si vous avez utilisé l'équation prédictive sur une créature, un objet ou un lieu au cours des derniers jours, vous pouvez apprendre un fait aléatoire sur le sujet qui est pertinent pour un sujet que vous désignez. Si vous possédez également la capacité de saveur magique Prémonition, une utilisation de l'une ou l'autre capacité vous accorde deux faits aléatoires mais liés sur le sujet. De plus, vous pouvez utiliser Modèle Prédictif sur le même sujet plusieurs fois (même si vous avez appris le niveau d'une créature), mais à chaque fois que vous le faites, vous devez appliquer un niveau d'Effort supplémentaire par rapport à votre utilisation précédente.
